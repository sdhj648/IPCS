Cloudflare Pages 与 Workers 快速部署文档

本款 IP/测速网站具备原生 Cloudflare 生态的高性能与低延迟集成兼容性

本项目完全实现了对 Cloudflare (Workers 和 Pages) 的完美适配。在 Cloudflare 生态上，本系统仅需利用系统自带的底层 request.cf 功能，即可秒级提取包括国家、城市、ASN、网速测试数据在内的全套核心指标。下面为您准备了极速上线的部署指令与配置说明。

极速部署指南 (Workers Cli)

# 1. 在本地电脑安装 Cloudflare Wrangler

npm install -g wrangler

# 2. 授权登录您的 Cloudflare 账户

wrangler login

# 3. 在项目根目录下直接部署到全球边缘网络

wrangler deploy

Cloudflare 自愈特性与环境变量

  • 免费边缘数据库解析: 免除任何 IPinfo 或 GeoIP 商业密钥，充分利用 Cloudflare 地理网格能力。
  • Pages 静态托管: 可以通过 Cloudflare Dashboard 控制面板一键导入 Github 存储库进行免费高精 Pages 发布。
